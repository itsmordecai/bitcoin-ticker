## The Goal

The objective was to learn about using Cupertino and Material Widgets in parallel and providing a different user interface depending on the platform.

## What I Have Created

A cryptocurrency price checking app. The app can be used to monitor the bitcoin (and other cryptocurrencies) value on the move!

![Finished App](https://github.com/londonappbrewery/Images/blob/master/bitcoin-flutter-demo.gif)

## What I Have Learnt

- How to use the DropdownButton Widget from Material design.
- How to loop through code using Dart for and for-in loops.
- How to use Cupertino Widgets in your app.
- How to check the platform your app is being run on to customise the UI for that platform.
